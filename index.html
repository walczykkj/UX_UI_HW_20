Vostok-SOS

Overview

Redesigning the website for Ukranian NGO Vostok-SOS to improve English-speaking user experience and make the donation process easier to use.

My Role

UX Designer

Research

Stakeholder

  • Goals of English language website vs. original Ukrainian language website are different
  • English website to focus on providing information to inspire donation
  • Would like to to improve donation user flow
  • Generate more traffic from outside Ukraine

Users

  • Completed 5 user interviews on Zoom
  • Explored preferences and behaviours when it comes to visiting NGO websites
  • Focused on the donation process and what inspires users to donate since this is the main goal of Vostok-SOS

Definition

Problem

The current English language version of the Vostok-SOS website was not serving the stakeholder’s intended purpose well – to provide users outside of Ukraine with information about their cause and solicit donations.

Our goal was to redesign the homepage for a non-Ukrainian audience, improve navigation and create a clear and easy-to-access call to action for donation

Prototyping

Since the donation user flow was top priority for our stakeholders, we started by improving and simplifying the existing user flow to highlight the most popular donation methods (PayPal and Credit Card) as the first thing users see, while still making other methods accessible in other tabs.

Testing

A/B Testing of Hero Image

We conducted A/B tests to determine which hero image was most impactful to users. The image of the masked soldier had the biggest emotional impact on 90% of users so we decided to lead with this image.
